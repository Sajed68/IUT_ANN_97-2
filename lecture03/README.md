## جلسه سوم
در این جلسه خواهیم دید:


* ایجاد لیست‌ها
* اندیس گذاری و برش لیست  
* متدهای پایه‌ای  
* لیست‌های تودرتو  


* ایجاد یک چندتای  
* چند متد پایه‌ای  
* خاصیت تغییرناپذیری  
* کی از چندتایی استفاده کنیم!
